# 주제

패션, 음악, 인터넷 문화 콘텐츠를 한곳에 아카이빙하여 사용자가 다양한 플랫폼과 콘텐츠를 탐색할 수 있도록 제작한 개인 취향 기반 아카이브 홈페이지이다.

HTML5와 CSS3를 활용하여 구현하였으며, Figma를 통해 UI를 설계한 후 Visual Studio Code에서 웹페이지로 제작하였다.

---

# 주제 선택 배경

평소 패션과 음악, 인터넷 문화에 관심이 많아 다양한 플랫폼과 콘텐츠를 자주 탐색해왔다. 하지만 정보가 여러 사이트와 SNS에 흩어져 있어 한눈에 정리하기 어렵다는 불편함을 느꼈다.

그래서 자주 참고하는 패션 플랫폼, 유튜브 채널, 음악 및 문화 콘텐츠를 하나의 공간에 정리할 수 있는 개인 아카이브 형태의 홈페이지를 제작하게 되었다.

또한 단순한 정보 나열이 아닌, 사용자의 취향과 감성을 보여줄 수 있는 UI를 구현하고자 하였으며, Figma를 활용한 UI 설계와 HTML/CSS 기반 웹페이지 제작 경험을 쌓고자 본 주제를 선택하였다.

---

# 사용된 주요 기술

## 1) HTML5

HTML5를 사용하여 웹페이지의 전체 구조를 설계하였다.

- `header`
- `nav`
- `section`
- `footer`

등의 시맨틱 태그를 활용하여 영역을 구분하였으며, 콘텐츠 구조를 직관적으로 구성하였다.

또한 `a` 태그를 사용하여 외부 사이트와 연결되는 링크 기능을 구현하였고, `img` 태그를 활용하여 다양한 이미지 콘텐츠를 배치하였다.

---

## 2) CSS3

CSS3를 활용하여 웹페이지의 전체 디자인과 레이아웃을 구현하였다.

Flexbox와 Grid 레이아웃을 사용하여 카드형 UI를 구성하였으며, 콘텐츠를 일정한 간격으로 정렬하였다.

또한 `border-radius` 속성을 사용하여 카드와 이미지에 둥근 모서리를 적용하였고, hover 효과 구현을 위해 `transition`과 `transform` 속성을 사용하였다.

이를 통해 사용자가 마우스를 올렸을 때 자연스럽게 확대되는 인터랙션 효과를 구현하였다.

---

## 3) 반응형 웹 구현

`media query`를 사용하여 화면 크기에 따라 레이아웃이 변경되도록 반응형 웹 구조를 구현하였다.

모바일 환경에서는 Grid 구조를 1열로 변경하여 작은 화면에서도 콘텐츠가 자연스럽게 배치되도록 설계하였다.

---

## 4) 웹폰트 적용

웹페이지의 전체적인 디자인 완성도를 높이기 위해 Pretendard 웹폰트를 적용하였다.

외부 CSS 파일을 불러온 후 전체 홈페이지에 적용하였으며, 이를 통해 기본 브라우저 폰트보다 깔끔하고 현대적인 분위기를 표현하였다.

---

## 5) Hover 인터랙션 효과

사용자 경험을 향상시키기 위해 카드형 콘텐츠에 hover 효과를 적용하였다.

마우스를 올렸을 때 이미지가 확대되고 카드가 위로 이동하도록 구현하였으며, `transition` 속성을 활용하여 자연스러운 애니메이션 효과를 적용하였다.

---

## 6) Figma 활용 UI 설계

홈페이지 제작 전 Figma를 사용하여 전체 UI와 레이아웃을 먼저 설계하였다.

카드 배치, 이미지 비율, 여백 구성 등을 시각적으로 설계한 후 HTML과 CSS를 활용하여 실제 웹페이지로 구현하였다.

---

## 7) AI 기반 도구 활용

프로젝트 제작 과정에서 AI 기반 도구를 활용하여 일부 레이아웃 아이디어와 CSS 구조를 참고하였다.

이를 바탕으로 HTML5와 CSS3 코드를 직접 수정 및 구현하며 최종 웹페이지를 완성하였다.

# 고찰

이번 프로젝트를 진행하면서 처음으로 Figma를 사용해보았는데, 실제로 사용해보니 홈페이지의 전체 구조와 레이아웃을 설계하는 데 큰 도움이 되었다.

코드를 직접 작성하지 않아도 hover 효과나 anchor 연결과 같은 다양한 인터랙션을 미리 적용해볼 수 있어서 디자인 작업을 훨씬 직관적으로 진행할 수 있었다.

특히 카드형 UI와 콘텐츠 배치를 먼저 시각적으로 설계한 뒤 HTML과 CSS로 구현하는 과정이 인상 깊었고, 디자인과 개발이 서로 연결되어 있다는 점을 느낄 수 있었다.

또한 Figma에서 hover 효과를 구현하려고 시도했지만 원하는 방식으로 잘 적용되지 않아, 해당 부분은 AI 기반 도구의 도움을 참고하여 CSS3의 hover 효과로 직접 구현하였다.

프로젝트를 시작하기 전에는 내가 직접 웹사이트를 만들 수 있을 것이라고 생각하지 못했고, 웹 개발이 어렵고 막막하게 느껴졌다. 하지만 이번 과제를 진행하면서 하나씩 직접 구현해보니 생각보다 재미있었고, 결과물을 완성하면서 자신감도 얻을 수 있었다.

또한 단순히 기술 자체만 중요한 것이 아니라, 어떤 아이디어와 방향성을 가지고 콘텐츠를 구성하는지가 더욱 중요하다는 점을 느꼈다.

AI의 도움을 적절하게 활용하니 어려운 부분도 훨씬 수월하게 해결할 수 있었고, 이를 통해 다양한 아이디어를 실제 결과물로 구현할 수 있다는 점이 인상 깊었다.
